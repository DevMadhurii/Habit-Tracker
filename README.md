
# Habit Tracker

A responsive web application designed to help users build consistency by tracking daily habits and maintaining streaks over time.

## Features
- Add and remove habits
- Track daily progress
- Maintain streak counts
- Responsive and user-friendly design
- Local storage support

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This project was built to encourage productivity and discipline while showcasing frontend development skills.

## Future Improvements
- User authentication
- Cloud data sync
- Habit analytics dashboard
